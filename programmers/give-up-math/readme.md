## 두번쨰 풀기
- 다른 사람의 풀이를 보고 진행하였음.
- 학생들의 답안지 array를 생성하는 것이 아니라 index를 변환해서 하는 방식으로 바꾸었음. 이렇게 하니 훨씬 깔끔.
- Max 값을 구할 때, Math.max() 를 사용하면 더 간결함. 나는 버블정렬을 하였음. Stream-reverseSort-max(1) 로 하니깐 실패함.
- `List<Integer>`를 int[] 로 변환하려면, mapToInt를 사용.
